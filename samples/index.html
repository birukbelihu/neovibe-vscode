Installation

  1. Open the Extensions panel in Visual Studio Code
    • Ctrl+Shift+X (Windows/Linux)
    • Cmd+Shift+X (macOS)
  2. Search for NeoVibe.
  3. Install & select either the Dark or Light NeoVibe theme.
Install from Visual Studio Code Marketplace
